# Chat Depth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the shell's CHAT destination into a real chat app: Room-backed persistent multi-conversation history, a hybrid HTTP/in-process transport, generation controls, markdown rendering, in-chat model switching with reload UX, and Markdown share/export.

**Architecture:** Persist conversations/turns in the existing `RelaisDatabase` (Room, v4→v5). A `ChatTransport` interface with two impls (`HttpChatTransport` via the already-present Ktor `SSE` client to the loopback `/v1/chat/completions`, and `InProcessChatTransport` wrapping `RelaisEngine.generate`) is chosen by a pure `chooseTransport` fn off a `GET /health` probe. A `ChatViewModel` owns the streaming lifecycle and observes Room `Flow`s; the existing amber `ChatScreen` is rebuilt to render markdown (reused `MarkdownText`/`BufferedFadingMarkdownText`), generation controls, a conversation drawer, in-chat model switch, and share/export.

**Tech Stack:** Kotlin, Jetpack Compose, Room (2.7.1), Ktor client (3.4.3, `Android` engine + `SSE` plugin), `com.halilibo.compose-richtext`, `org.json`, JUnit + Robolectric (device-free unit tests in `test/`).

```kotlin
import androidx.room.Entity
import androidx.room.ForeignKey

@Entity(tableName = "conversations")
data class Conversation(
  @PrimaryKey val id: String,
  val title: String,
  val modelId: String,
  val createdAt: Long,
  val updatedAt: Long,
)

@Entity(
  tableName = "chat_turns",
  foreignKeys = [
    ForeignKey(
      entity = Conversation::class,
      parentColumns = ["id"],
      childColumns = ["conversationId"],
      onDelete = ForeignKey.CASCADE,
    )
  ],
  indices = [Index("conversationId"), Index(value = ["conversationId", "createdAt"])],
)
data class ChatTurn(

  val conversationId: String,
  val role: String,            // "user" | "assistant"
  val content: String,
  val attachmentType: String?, // "image" | "audio" | null
  val attachmentPath: String?, // app-storage path, or null
  val answeredByModelId: String?,
  val answeredByBackend: String?,
  val createdAt: Long,
)
```

```kotlin
package cc.grepon.relais.data

import androidx.room.Dao
import androidx.room.Insert
import androidx.room.OnConflictStrategy
import androidx.room.Query
import kotlinx.coroutines.flow.Flow

@Dao
interface ChatDao {
  @Insert(onConflict = OnConflictStrategy.REPLACE)
  suspend fun upsertConversation(c: Conversation)

  @Query("SELECT * FROM conversations ORDER BY updatedAt DESC")
  fun observeConversations(): Flow<List<Conversation>>

  @Query("SELECT * FROM chat_turns WHERE conversationId = :conversationId ORDER BY createdAt ASC, id ASC")
  fun observeTurns(conversationId: String): Flow<List<ChatTurn>>

  @Query("SELECT * FROM chat_turns WHERE conversationId = :conversationId ORDER BY createdAt ASC, id ASC")
  suspend fun turnsFor(conversationId: String): List<ChatTurn>

  @Insert(onConflict = OnConflictStrategy.REPLACE)
  suspend fun insertTurn(t: ChatTurn)

  @Query("UPDATE conversations SET title = :title, updatedAt = :updatedAt WHERE id = :id")
  suspend fun renameConversation(id: String, title: String, updatedAt: Long)

  @Query("DELETE FROM conversations WHERE id = :id")
  suspend fun deleteConversation(id: String)

  @Query("DELETE FROM chat_turns WHERE conversationId = :conversationId AND createdAt > :afterCreatedAt")
  suspend fun deleteTurnsAfter(conversationId: String, afterCreatedAt: Long)
}
```
